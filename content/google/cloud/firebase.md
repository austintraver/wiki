# Command Line

* [Link to documentation](https://firebase.google.com/docs/cli)

* Installing Firebase CLI:

  ```shell
  curl -sL https://firebase.tools | bash
  ```

* Log into firebase

  ```shell
  firebase login
  ```

* Set the active project

  ```shell
  firebase use project_id
  ```
