# Installation

First, install the Azure command line utilities

* On macOS

  ```sh
  brew install azure-cli
  ```

* On Ubuntu

  ```sh
  curl -sL https://aka.ms/InstallAzureCLIDeb | sudo bash
  ```

Next, log into your account

```sh
az login
```
