# Google Docs

* Share a link to a document as a PDF

  * Append `/export?format=pdf`

  * Before

    ```txt
    https://docs.google.com/document/d/12345abcde/
    ```
  * After

    ```txt
    https://docs.google.com/document/d/12345abcde/export?format=pdf
    ```
